# CATE 2026 Workshop Site

Static GitHub Pages starter for CATE 2026: Cryptography for AI and Trustworthy Ecosystems.

The GitHub repository is:

```text
git@github.com:cateworkshop/cateworkshop.github.io.git
```

This local checkout uses the equivalent SSH alias URL so Git selects the dedicated `cateworkshop` key:

```text
git@github.com-cateworkshop:cateworkshop/cateworkshop.github.io.git
```

The custom SSH host `github.com-cateworkshop` is configured in `~/.ssh/config` to use `~/.ssh/id_ed25519_cateworkshop`.

## Publish

Push this repository's `main` branch to `origin`. GitHub Pages should serve the site at `https://cateworkshop.github.io/`.

## Local Preview

Open `index.html` directly in a browser. No build step is required.
